## 📋 О ПРОЕКТЕ

**Простой и понятный Telegram-бот для учета посещений тренировок в боксерском клубе.**  
Идеально подходит для тренеров, которым нужен **быстрый и надежный** инструмент без лишних функций.

### 🎯 ДЛЯ КОГО ЭТОТ БОТ?

- 👨‍🏫 **Персональные тренеры** по боксу
- 🥊 **Боксерские клубы** с несколькими тренерами
- 💪 **Фитнес-клубы** с групповыми тренировками

### ✨ ВОЗМОЖНОСТИ

| Функция | Описание |
|---------|----------|
| ✅ **Мульти-тренерский режим** | Несколько тренеров работают в одном боте |
| 📋 **Учет учеников** | Добавление, группы, остаток занятий |
| 🎯 **Отметка посещений** | Быстрая отметка всей группы |
| 💰 **Продажа абонементов** | 6 видов абонементов с ценами |
| 🎨 **Цветовая индикация** | 🔴 мало занятий, 🟡 нормально, 🟢 много |
| 🔄 **Смена тренера** | Если ошиблись при входе |
| ➕ **Восстановление занятий** | По болезни или особым случаям |

### 🚀 ПРЕИМУЩЕСТВА

- ✅ **Простота** - разберется любой тренер за 5 минут
- ✅ **Надежность** - никаких сложных багов
- ✅ **Скорость** - отметка группы за 30 секунд
- ✅ **Понятность** - минимум кнопок, максимум пользы

## 🛠 УСТАНОВКА

### 1. Клонировать репозиторий
```bash
git clone https://github.com/yourusername/mr-magic-bot-simple.git
cd mr-magic-bot-simple
```
Установить зависимости

```
pip install -r requirements.txt

```

Настроить токен бота
Создайте файл .env и добавьте:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

Запустить бота

```
python bot/main.py
```



КАК ПОЛЬЗОВАТЬСЯ
👋 Первый вход
Напишите боту /start

Выберите своё имя из списка тренеров

Или нажмите "➕ Новый тренер", чтобы добавить себя

👤 Добавление ученика
Нажмите "➕ Добавить ученика"

Выберите группу (Baby-бокс, Взрослые и т.д.)

Введите имя ученика

Введите телефон (или пропустите)

Выберите абонемент

✅ Отметка посещений
Нажмите "✅ Отметить посещение"

Выберите группу

Отмечайте учеников:

✅ Был - занятие списывается

❌ Не был - занятие ТОЖЕ списывается (сгорает)

В конце покажут подробную статистику

💰 Продажа абонемента
Нажмите "💰 Продать абонемент"

Выберите ученика

Выберите тип абонемента

Занятия автоматически добавятся

⚙️ Настройки
🔄 Сменить тренера - если ошиблись при входе

➕ Добавить нового тренера - добавить коллегу
